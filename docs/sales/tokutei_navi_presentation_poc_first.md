# 特定技能2号移行ナビ — 面談プレゼンテーション（PoC提案中心版）

> このファイルは `docs/sales/tokutei_navi_presentation.md`（プロダクト紹介中心版）に対する**代替構成案**です。現行版は削除・変更せずそのまま保持し、本ファイルを比較検討用の別バージョンとして新規作成しています。どちらを正本とするかは、内容を比較した上で別途決定してください。
>
> 来週の日本企業面談では「PoC提案」が主目的であるため、本バージョンは**企業課題の直後にPoC提案を前倒しする構成**を採用しています。各スライドには、現行版との対応関係と変更理由を付記し、比較しやすくしています。
>
> 全8枚構成。各スライドは「タイトル／掲載内容／話す内容／表示するLovable画面／企業側に伝えたいメッセージ／変更理由（現行版との比較）」の6項目で記述しています。
>
> 本資料は営業資料であり、断定表現・制度判断は行わず、「可視化」「初期診断」「棚卸し」「PoC」「専門家確認」を中心語彙として統一しています。

## Slide3：PoC提案

| 項目 | 内容 |
|---|---|
| タイトル | まずは5〜20名でPoCから |
| 掲載内容 | PoCの位置づけ（大規模導入の前段階としての低負担な初期診断実施）、対象規模5〜20名、**「これは初期診断であり、在留資格変更許可を保証するものではない」旨をこの時点で明記** |
| 話す内容 | 「本日ご提案したいのは、いきなりの全社導入ではありません。まずは5〜20名程度を対象に、在留期限・日本語レベル・管理者相当経験を初期診断として可視化するPoCから始めることをご提案します。診断結果はあくまで初期診断であり、在留資格変更許可を保証するものではない点は、最初にお伝えしておきます」 |
| 表示するLovable画面 | なし |
| 企業側に伝えたいメッセージ | 今日の面談の依頼事項を早期に明確にする。低リスク・低負担な依頼であることを伝え、警戒感を下げる |
| 変更理由（現行版との比較） | 現行版では全8枚中6枚目に登場していたPoC提案を3枚目へ前倒し。面談の主目的が「PoC提案」に変わったため、依頼内容を早い段階で明示し、時間切れによる説明不足や「結局何を依頼されているか分からない」という状態を防ぐ。免責注記もこのタイミングに前倒しして記載する必要がある（現行版では4枚目の診断画面まで登場しなかった） |

| 企業側に伝えたいメッセージ | PoCへの協力が「何かをもらえる」具体的なリターンに直結することを示す。依頼だけでなく提供物を明確にすることで合意形成を後押しする |

## PoC実施条件（対象企業・対象人数・前提条件）

| 区分 | 内容 |
|---|---|
| 対象企業 | 外食業法人で、特定技能1号人材を5名以上雇用している企業（`docs/sales/tokutei_navi_one_pager.md`の対象条件と整合） |
| 対象人数 | 初回PoCは5〜20名（`docs/poc/tokutei_navi_poc_design.md`の対象人数と同一基準） |
| 前提条件 | ・対象人材の在留期限、現在の日本語レベル、管理者相当実務経験の開始日を企業側で準備できること<br>・個人情報の取り扱いについて社内確認が済んでいること（氏名の代わりに社員番号での匿名運用が可能）<br>・診断結果は初期診断であり、在留資格変更許可を保証するものではないことに同意いただけること<br>・必要に応じて行政書士・登録支援機関への接続を前提とすること |

本条件は、Slide3（PoC提案）およびSlide8（次アクション）で言及する内容の詳細版です。面談当日はSlide3で概要のみ伝え、本条件表は面談後の持ち帰り資料・ヒアリングシート記入時の参照情報として使用することを想定しています。
