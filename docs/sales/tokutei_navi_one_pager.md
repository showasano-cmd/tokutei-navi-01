# 特定技能2号移行ナビ — 1枚概要資料

> 面談後に企業担当者が社内共有できる要約資料です。本ツールは在留資格変更の可否を保証するものではなく、初期診断・リスク整理・企業ヒアリング・PoC設計のための補助ツールです。

---

## プロダクト名

**特定技能2号移行ナビ**（Tokutei Gino 2-go Migration Navigator）

外食業 法人向け 特定技能2号移行可能性の初期診断ツール

---

## 対象

| 対象 | 想定される利用シーン |
|---|---|
| 外食業 法人（人事・店舗管理担当） | 自社在籍の特定技能1号人材の棚卸し・早期リスク把握 |
| 人材紹介・送り出し関連会社 | 送客先企業への移行状況の可視化支援 |
| 登録支援機関 | 支援対象者の移行タイムライン管理・優先度整理 |

PoCの初回対象は、まずは特定技能1号人材を5名以上雇用している企業を想定しています。

---

## よくある課題

- 特定技能1号人材の在留期限が2025〜2026年に集中到来しているが、社内で一元管理できていない
- 「誰が」「いつまでに」「何をすべきか」を把握している担当者が不在、または属人化している
- N3取得・技能試験・管理者相当実務経験の3条件が同時進行で必要なことが現場に浸透していない
- 2号移行が間に合わない場合の代替手段（80%特例ルート等）の検討が後手に回りやすい
- 移行判断のための投資対効果（採用費・育成費）が可視化されていない

---

## このツールで分かること

在留期限・日本語レベル・管理者相当実務経験の開始日という3つの情報から、対象人材ごとに以下を**初期診断として可視化**します。

- 在留期限までの残り期間と、そこから逆算した現在地
- N3取得の有効な受験機会があるかどうか（合否発表・申請準備期間を考慮）
- 技能試験の受験機会回数の見込み
- 管理者相当実務経験の充足状況
- 3条件を統合した総合的な状況感（○ / △ / ✕ の3段階）
- 各条件が試験スケジュールとどう噛み合うかの時系列タイムライン
- 通常ルートでの移行が厳しい場合に確認すべき80%特例ルートの論点
- これまでの採用・育成投資の概算と、移行できなかった場合のリスク
- 今すぐ着手すべきアクションの優先順位

---

## 入力項目

| 項目 | 内容 |
|---|---|
| ① 在留期限の満了日 | 在留カード記載の期限日 |
| ② 現在の日本語レベル | J.TEST Band または JLPT 取得状況 |
| ③ 管理者相当の実務経験 開始日 | 副店長・サブリーダー等の辞令発行日 |
| （任意）氏名または社員番号 | 社内共有・印刷時の識別用。空欄可。社員番号での匿名運用も可能。 |
| （任意）採用費用・月次コスト | 投資対効果の概算に使用（未入力時は業界概算値を使用） |

---

## 出力内容

| セクション | 内容 |
|---|---|
| 総合判定 | ○ / △ / ✕ の3段階（初期診断としての目安） |
| 現在地と残り期間 | 在留期限まで・N3有効機会・実務経験月数 |
| 3要件チェック | 各要件の充足状況と注意点 |
| 移行タイムライン | JLPT・技能試験・実務経験の時系列照合 |
| 80%特例ルート | 必要な場合のみ表示される確認事項 |
| 投資対効果 | 採用・育成コストの可視化 |
| 今すぐ取るべきアクション | 優先順位付きの行動リスト |

診断結果はコピー・印刷（PDF保存）に対応しており、社内共有資料としてそのまま利用できます。

---

## PreCheck / インテグリティ診断との接続

本ツールは、単独の診断で完結するのではなく、次のステップにつながる法人向けファネルの入口として設計されています。

```
① tokutei-navi-01（本ツール）
   在留期限・3要件から「移行タイムラインに間に合うか」を初期診断
        ↓
② PreCheck（日本語能力診断）
   N3〜N4相当の日本語能力の現在地を短時間で確認
        ↓
③ tokutei-integrity-01（インテグリティ診断）
   管理職素養・投資継続可否を判断材料として提供
```

診断結果画面下部から、PreCheckおよびインテグリティ診断への導線を設けています。3ツールを通じて「間に合うか」→「日本語能力は十分か」→「投資を継続すべきか」という順で、企業の意思決定を段階的に支援します。

## PoC提案

本ツールは、大規模な導入検討の前に、少人数（5〜20名程度）を対象とした**PoC（概念実証）から始めることを想定**しています。

- 対象人材の在留期限・日本語レベル・実務経験開始日を用意いただく
- 本ツールで一括して初期診断を実施し、移行候補者の棚卸しとリスクの可視化を行う
- 結果をもとに、優先対応が必要な人材・要専門家確認の人材を整理する
- 詳細は `docs/poc/tokutei_navi_poc_design.md` を参照

## 注意書き

- 本ツールは、在留期限・日本語レベル・実務経験の情報をもとにした**初期診断**であり、在留資格変更許可を保証するものではありません。
- 診断結果は、企業内での状況整理・優先順位付け・専門家への相談準備を目的とした補助情報です。
- 実際の申請可否は、最新の公的情報、本人の在留状況、雇用契約、実務経験証跡、試験結果などをもとに、行政書士・登録支援機関等の専門家による確認が必要です。
